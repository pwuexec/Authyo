# Authy - Identity and Access Management System

Authy is a comprehensive Identity and Access Management (IAM) system designed to manage users, organizations, roles, and permissions securely.

## Documentation

### Stakeholders

For a detailed overview of the system's current features and future roadmap, please refer to the **[Product Capabilities](docs/stakeholders/PRODUCT_CAPABILITIES.md)** document. This guide is tailored for business stakeholders and provides insights into:

*   **Implemented Capabilities**: Authentication, Session Management, Organization Management, and RBAC.
*   **Planned Capabilities**: Upcoming features such as Secure User Registration, Advanced Organization Lifecycle, and Granular Access Control.
